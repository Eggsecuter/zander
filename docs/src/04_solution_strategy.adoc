== Solution Strategy

Der gewählte Lösungsansatz sieht eine modular aufgebaute Python-Software vor, welche die Bildverarbeitung, Entscheidungslogik und Kommunikation klar voneinander trennt. Mithilfe von OpenCV werden Puzzleteile erkannt und deren Lage bestimmt, während eine regelbasierte Logik die Puzzle-Lösung berechnet. Diese Struktur ermöglicht eine gute Wartbarkeit, einfache Erweiterbarkeit und gezielte Tests, insbesondere durch die Anbindung einer Simulationsumgebung.
